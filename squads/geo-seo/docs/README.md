# GEO/SEO Squad

## Missão
Auditar e optimizar a visibilidade de sites em pesquisas IA (ChatGPT, Perplexity, Google AI Overviews, Gemini, Claude) e pesquisa tradicional, usando frameworks de 6 especialistas de elite mundial.

## Quando Usar
- Auditar um site para AI search readiness
- Gerar diagnóstico GEO completo
- Optimizar sites para visibilidade IA
- Analisar concorrência em AI search
- Gerar plano de acção prioritizado

## Como Activar
```
@geo-seo-chief
```

## Comandos Principais

| Comando | Descrição |
|---------|-----------|
| `*audit {url}` | Auditoria GEO completa |
| `*score {url}` | Apenas GEO Score (0-100) |
| `*action-plan {url}` | Plano de acção prioritizado 30/60/90 dias |
| `*shopify-audit {url}` | Auditoria específica para Shopify |
| `*compare {url1} {url2}` | Comparar dois sites |
| `*help` | Todos os comandos |
