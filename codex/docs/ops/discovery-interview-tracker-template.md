## Regras de preenchimento

- `status_dor = validada` somente se frequencia >= 4 e intensidade >= 4.
- `impacto_estimado` deve ser numerico ou faixa concreta (ex: "R$ 15k/mes").
- `frase_chave` deve conter uma citacao literal curta do entrevistado.
