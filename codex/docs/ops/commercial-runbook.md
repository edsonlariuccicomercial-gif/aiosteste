# Runbook Comercial

Data: 03/03/2026

## Objetivo
Gerar rapidamente um kit comercial pronto para abordagem de fornecedores.

## Comando unico
```powershell
npm.cmd run commercial:kit
```

## Saidas
- `docs/ops/commercial-kit.json`
- `docs/ops/commercial-kit.md`

## Uso recomendado
1. Rodar antes de cada bloco comercial do dia.
2. Usar `commercial-kit.md` como roteiro de call.
3. Registrar ajustes de plano em `docs/strategy/commercial-packaging-plan.md`.
