# Story 1.5 - Base de Historico de Precos para Sugestao de Lance

## Status
Done

## Objetivo
Consolidar uma base historica de precos por objeto/SKU/regiao para apoiar sugestao de lance com referencia real de mercado no fluxo diario.

## Criterios de Aceite

- [x] Historico consolidado em arquivo unico com versao e data de atualizacao.
- [x] Cada registro contem no minimo: objeto normalizado, SKU, SRE/municipio, custo estimado, preco sugerido, data de coleta.
- [x] Script gera resumo estatistico por SKU (min/mediana/media/max).
- [x] Dashboard consome o historico para exibir faixa de referencia de preco.
- [x] Validacao detecta registros invalidos/incompletos no historico.

## Tarefas

- [x] Definir schema da base historica de precos.
- [x] Criar script de consolidacao a partir de `quotes.json`.
- [x] Criar script de estatisticas por SKU e regiao.
- [x] Integrar faixa de referencia no dashboard (simulador/tabela).
- [x] Adicionar validacao automatica da integridade da base historica.

## Riscos

- Baixo volume inicial pode gerar referencia distorcida por SKU/regiao.
- Dados de objeto mal normalizados podem contaminar estatisticas.
- Mudancas no coletor podem quebrar retrocompatibilidade do historico.

## Mitigacoes

- Exibir tamanho da amostra por faixa de preco.
- Reusar normalizacao de objeto/SKU ja consolidada no modulo.
- Versionar schema da base e validar antes de publicar artefato.
