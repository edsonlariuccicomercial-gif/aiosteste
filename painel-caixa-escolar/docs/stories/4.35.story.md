## Story

**As a** operador que precisa atender pedidos de contratos,
**I want** cadastrar produtos reais do mercado com suas equivalências às descrições de contrato,
**so that** o sistema saiba qual produto comprar quando receber um pedido.

---

## Contexto / Motivação

O fluxo pós-licitação da Caixa Escolar é:

```
Contrato → Pedido (da escola) → Nota Fiscal → Demanda → Estoque / Compras
```

O gap crítico está logo no início: quando uma escola faz um pedido referenciando a descrição do contrato ("PAPEL TOALHA INTERFOLHADO 100% CELULOSE VIRGEM 23X21CM"), o sistema precisa saber que isso corresponde ao produto real de mercado "Papel Toalha Mili cx c/ 1000fls" para calcular quantidades de compra, movimentar estoque e gerar listas de compras.

Hoje, os itens em `bancoPrecos.itens[]` são preços importados sem identidade de produto estruturada. Esta story os evolui para registros de produto completos com vínculo às descrições de contrato.

### Estrutura existente que será evoluída

- `bancoPrecos.itens[]` — array de itens de preço (campos: nome, preco, unidade, fornecedor, data)
- `itensMestres[]` — índice de normalização de produtos (Story 4.26)
- `findBestMestre(nome)` — retorna `{ mestre, score }` usando aliases + similaridade
- `calcSimilarity(a, b)` — similaridade token-based Jaccard
- `UNIT_CONVERSIONS` — conversão básica de embalagens

// Valor:

### Funções principais

```javascript
const GDP_EQUIV_KEY = 'gdp.equivalencias.v1';

function loadEquivalencias() {
  return JSON.parse(localStorage.getItem(GDP_EQUIV_KEY) || '{}');
}

function saveEquivalencias(equiv) {
  localStorage.setItem(GDP_EQUIV_KEY, JSON.stringify(equiv));
}

function getEquivalencia(descricao) {
  const equiv = loadEquivalencias();
  return equiv[normalizedText(descricao)] || null; // retorna SKU ou null
}

function setEquivalencia(descricao, sku) {
  const equiv = loadEquivalencias();
  equiv[normalizedText(descricao)] = sku;
  saveEquivalencias(equiv);
  // re-renderizar itens afetados — disparar evento ou chamar render diretamente
}

function getProdutoBySku(sku) {
  return bancoPrecos.itens.find(i => i.sku === sku) || null;
}
```

### Auto-sugestão no modal

```javascript
function abrirModalVincular(descricao) {
  _descricaoVinculando = descricao;
  document.getElementById('modal-vincular-titulo').textContent =
    `Vincular: ${descricao.slice(0, 60)}...`;

  // auto-sugestão
  const { mestre, score } = findBestMestre(descricao);
  if (score >= 0.5) {
    renderSugestaoAutomatica(mestre, score);
  }

  // resultados iniciais: todos os produtos

  abrirModal('modal-vincular-produto');
}
```
