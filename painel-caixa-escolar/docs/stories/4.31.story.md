## Story

**As a** operador que consulta preços via PNCP na aba Oportunidades,
**I want** que a pesquisa de preço retorne resultados corretamente,
**so that** eu consiga ver o menor preço praticado em licitações para cada item do orçamento.

---

## Contexto / Motivação

A funcionalidade de pesquisa de preço no PNCP está quebrada por dois bugs distintos introduzidos em refatorações anteriores:

**Bug 1 — Field mismatch em `pesquisarPrecoPNCP()`:**
A função `pesquisarPrecoPNCP()` (handler do botão de pesquisa) lê `resultado.menorPreco` e `resultado.orgao`, mas a função `consultarPNCP()` retorna `resultado.min` e o orgão dentro de `resultado.detalhes[0].orgao`. Os campos nunca batem, então a UI exibe sempre vazio ou NaN.

**Bug 2 — Endpoint inexistente:**
Ao consolidar as Vercel functions para respeitar o limite de 12 funções, o arquivo `api/pncp-search.js` foi removido. O rewrite `/.netlify/functions/pncp-search` → `/api/pncp-search` foi atualizado em `vercel.json`, mas o arquivo de destino não existe mais. A solução correta é adicionar a ação PNCP ao proxy unificado `api/caixa-proxy.js`.

**Melhoria adicional — Termo de busca muito específico:**
O nome completo do item SGD é enviado ao PNCP como termo de pesquisa (ex: "CADERNO CAPA DURA 96 FOLHAS PAUTADAS"). Isso é específico demais e retorna zero resultados. O sistema deve simplificar automaticamente o termo: pegar as 3 primeiras palavras significativas (ignorando preposições e artigos como "de", "da", "do", "para", "com", "e", "a", "o", "em").

### Função de simplificação do termo

```javascript
function simplificarTermoPNCP(nomeItem) {
  const stopwords = new Set([
    'de', 'da', 'do', 'dos', 'das', 'para', 'com', 'e', 'a', 'o', 'em',
    'por', 'ou', 'ao', 'aos', 'as', 'um', 'uma', 'uns', 'umas', 'se'
  ]);
  const palavras = nomeItem
    .toUpperCase()
    .split(/\s+/)
    .filter(p => p.length > 1 && !stopwords.has(p.toLowerCase()));
  return palavras.slice(0, 3).join(' ');
}
```

### Adicionar ação ao caixa-proxy.js

```javascript
// Dentro do handler do proxy, adicionar:
if (action === 'pncp-search') {
  const { termo, uf } = body;
  const url = `https://pncp.gov.br/api/search/?q=${encodeURIComponent(termo)}&uf=${uf || ''}&tam=10`;
  const r = await fetch(url, { headers: { 'Accept': 'application/json' } });
  const data = await r.json();
  // Parsear resposta e retornar no formato { min, detalhes }
  const detalhes = (data.data || []).map(item => ({
    orgao: item.orgaoEntidade?.razaoSocial || '',
    preco: parseFloat(item.valorUnitarioEstimado || 0),
    descricao: item.descricao || '',
    data: item.dataPublicacaoEdital || ''
  }));
  const min = detalhes.length > 0 ? Math.min(...detalhes.map(d => d.preco)) : null;
  return res.status(200).json({ min, detalhes });
}
```
