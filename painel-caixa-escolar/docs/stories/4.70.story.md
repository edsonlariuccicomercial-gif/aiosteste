## BLOCO A — Importacao de Planilha padrao (modelo CR)

O modelo de referencia e a importacao de Contas a Receber (`gdp-contratos.html:1634-1828`):
- Modal com drag & drop
- Auto-deteccao de colunas via fuzzy matching
- Preview com primeiras 8 linhas antes de confirmar
- Aceita .xlsx, .xls, .csv (SheetJS/XLSX)

### Acceptance Criteria — Bloco A

- [ ] **AC-A1:** Central de Produtos — substituir importacao antiga (Baixar Modelo + Importar Excel) pelo modal padrao CR com drag&drop, auto-deteccao de colunas e preview
- [ ] **AC-A2:** Fornecedores — adicionar botao "Importar Planilha" com modal padrao CR (colunas: razao social, CNPJ, telefone, email, cidade, UF)
- [ ] **AC-A3:** Contas a Pagar — adicionar botao "Importar Planilha" com modal padrao CR (colunas: descricao, categoria, forma, valor, vencimento, status)
- [ ] **AC-A4:** Criar funcao reutilizavel `createImportModal(config)` para evitar duplicacao de codigo — cada sessao passa config com colunas, funcao de mapeamento e funcao de save
