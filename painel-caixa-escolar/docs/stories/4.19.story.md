## Story

**As a** gestor administrativo,  
**I want** acompanhar as notas fiscais em uma aba dedicada e gerar cobranca automaticamente junto com a NF,  
**so that** eu consiga controlar autorizacao SEFAZ e cobranca no mesmo fluxo.

---

## Acceptance Criteria

### AC-1: Aba dedicada
- [ ] Deve existir aba `Notas Fiscais`
- [ ] A aba deve listar numero, pedido, cliente, valor, status fiscal e status de cobranca

### AC-2: Workflow SEFAZ
- [ ] A NF deve transitar pelos status `rascunho`, `pendente_autorizacao`, `autorizada`, `rejeitada`, `cancelada`
- [ ] O sistema deve manter campos para lote, protocolo e chave de acesso

### AC-3: Cobranca automatica
- [ ] Ao gerar NF, o sistema deve gerar tambem uma cobranca bancaria
- [ ] A forma padrao inicial deve ser boleto
- [ ] A cobranca deve nascer vinculada a `nota_fiscal_id`

### AC-4: Integracao futura
- [ ] O sistema deve deixar payload e status preparados para API bancaria e SEFAZ
- [ ] Mesmo sem integracao real, a modelagem deve ser clara e rastreavel
