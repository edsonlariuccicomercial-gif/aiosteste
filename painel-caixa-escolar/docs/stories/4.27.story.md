## Story

**As a** operador comercial que recebe tabelas de fornecedores a cada 15 dias,
**I want** que o sistema armazene o histórico completo de preços por fornecedor, por data e por arquivo de origem,
**so that** eu possa rastrear a evolução de preços, identificar tendências e tomar decisões baseadas em dados reais.

---

## Contexto / Motivação

O `bancoPrecos` atual tem `custosFornecedor: [{fornecedor, preco, data}]` — uma lista simples que perde contexto. Não rastreia:
- De qual arquivo veio o preço
- Qual a confiança da extração (PDF vs Excel vs OCR)
- Tendência temporal (subindo, caindo, estável)
- Se o preço foi atualizado ou é uma nova entrada

### Regras de negócio críticas (do prompt)

- **NUNCA sobrescrever dados históricos** — sempre append
- **Sempre armazenar descrição original** junto com a normalizada
- **Suportar múltiplas versões** de tabela por fornecedor
- **Rastreabilidade total**: de onde veio cada preço

## Acceptance Criteria

### AC-1: Registro de arquivos importados

- [ ] Cada importação registra um "arquivo fonte": `{ id, nomeArquivo, fornecedor, dataImportacao, tipoFonte, qtdItens, confianca }`
- [ ] Tipo de fonte: "excel" (confiança alta), "pdf-texto" (média), "pdf-ocr" (baixa), "manual" (alta)
- [ ] Salvar em `localStorage` com chave `caixaescolar.arquivos-importados`
- [ ] Exibir lista de importações recentes no Banco de Preços

### AC-2: Histórico de preços append-only

- [ ] Cada registro de preço contém: `{ preco, data, fornecedor, arquivoId, descricaoOriginal, confianca }`
- [ ] Ao importar um preço que já existe (mesmo fornecedor + mesmo item mestre), NÃO sobrescrever — adicionar ao histórico
- [ ] O campo `custoBase` do item do banco deve refletir o **último preço mais recente** (não o menor)
- [ ] Histórico acessível via expansão da linha ou modal

### AC-3: Indicador de tendência de preço

- [ ] Para cada item com >= 2 registros de preço, calcular tendência:
  - Subindo: último preço > penúltimo (badge vermelho com seta para cima)
  - Caindo: último preço < penúltimo (badge verde com seta para baixo)
  - Estável: variação < 5% (badge cinza)
- [ ] Exibir indicador de tendência na tabela do banco
- [ ] No hover/tooltip, mostrar: "R$ X → R$ Y (±Z%)"

### AC-4: Estatísticas por item mestre

- [ ] Para cada item mestre, calcular: menor preço, maior preço, média, mediana
- [ ] Identificar "fornecedor mais competitivo" (menor preço mais recente)
- [ ] Score de confiança ponderado (Excel > PDF > OCR)
- [ ] Exibir essas stats no modal de detalhes do item

### AC-5: Timeline visual de preços

- [ ] Ao clicar em um item do banco, abrir painel/modal com timeline de preços
- [ ] Lista cronológica: data | fornecedor | preço | fonte | confiança
- [ ] Highlight do menor preço em cada período
- [ ] Permitir filtrar por fornecedor específico

### AC-6: Detecção de variação significativa

- [ ] Ao importar, se um preço varia mais de 20% em relação ao último registro, alertar
- [ ] Toast: "Alerta: {item} variou {X}% (R$ Y → R$ Z) do fornecedor {F}"
- [ ] Logar no console para auditoria
