## Root Cause (Analise @analyst)

### Bug 1 — Saldo corrupto
Valor 8.82 foi salvo no Supabase ANTES do fix da Story 4.63. syncFromCloud() sobrescreve local com dado corrupto. Fix do codigo ja aplicado, falta corrigir o dado persistido.

### Bug 2 — Exclusoes reaparecem
3 problemas combinados:
1. `gdp.conciliacao.v1` em `SHARED_SYNC_KEYS` (app-state.js:643) — forca cloud overwrite
2. Protecao de merge so ativa quando local > cloud (app-sync.js:144-147) — delete faz local < cloud, protecao nao ativa
3. `DELETED_KEY_MAP` (gdp-api.js:138-153) nao tem entrada para conciliacao — sem tracking de deletes
