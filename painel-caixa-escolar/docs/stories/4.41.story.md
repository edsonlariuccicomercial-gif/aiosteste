## Story

**As a** operador que pesquisa preços de referência,
**I want** que o botão de pesquisa PNCP (lupa) no pré-orçamento funcione corretamente,
**so that** eu tenha preços de referência reais para minhas cotações.

## Contexto / Motivação

A Story 4.31 corrigiu 3 bugs do PNCP:
1. Field mismatch (`resultado.min` vs `menorPreco`) — corrigido
2. Missing proxy no painel-caixa-escolar (adicionado ao `caixa-proxy.js`) — corrigido
3. Termo de busca simplificado — corrigido

Mas o operador reporta que ainda não funciona quando clica na lupa. As causas prováveis a investigar:

- O `caixa-proxy.js` aceita `action: "search"/"items"`, e `consultarPNCP()` no `app.js` envia para `/.netlify/functions/pncp-search`, que faz rewrite para `/api/caixa-proxy`. O body tem `action: "search"` que o proxy mapeia para `handlePncpSearch`. Isso **deveria** funcionar — mas pode não estar.
- O rewrite `/.netlify/functions/pncp-search` → `/api/caixa-proxy` pode não estar configurado corretamente no `vercel.json` do projeto `painel-caixa-escolar`
- Pode ser timeout: a busca PNCP faz múltiplas chamadas (search + items para cada resultado)
- A API PNCP pode estar fora do ar ou ter mudado seu contrato

O dev deve investigar primeiro, depois corrigir.

## Dev Notes

### Implementação de timeout com AbortController

```javascript
async function consultarPNCP(termo) {
  const controller = new AbortController();
  const timeoutId = setTimeout(() => controller.abort(), 15000);

  console.log('[PNCP] buscando:', termo);

  try {
    const response = await fetch('/.netlify/functions/pncp-search', {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ action: 'search', termo }),
      signal: controller.signal
    });

    console.log('[PNCP] response status:', response.status);

    if (!response.ok) {
      throw new Error(`HTTP ${response.status}`);
    }

    const data = await response.json();
    console.log('[PNCP] resultados:', data?.resultados?.length ?? 0);
    return data;

  } catch (err) {
    if (err.name === 'AbortError') {
      showToast('PNCP: timeout — tente novamente', 'warning');
    } else if (err.message.startsWith('HTTP')) {
      showToast(`PNCP indisponível (${err.message}) — tente novamente`, 'error');
    } else {
      showToast('PNCP: erro de rede — verifique a conexão', 'error');
    }
    console.error('[PNCP] erro:', err);
    return null;
  } finally {
    clearTimeout(timeoutId);
  }
}
```
