## Story

**As a** operador comercial que pesquisa preços em sites de fornecedores,
**I want** colar a URL de um site de fornecedor e o sistema extrair automaticamente os produtos com preços,
**so that** eu alimente o banco de preços com dados de mercado real sem copiar manualmente item por item.

---

## Contexto / Motivação

O operador visita sites de fornecedores (atacadistas, distribuidores) que listam produtos com preços. Hoje ele copia manualmente para planilha e depois importa. Com IA (GPT via Vercel function), podemos:

1. Buscar o HTML da página via proxy serverless
2. Enviar o conteúdo para IA extrair dados estruturados
3. Apresentar preview para o usuário confirmar
4. Importar no banco com rastreabilidade

### Stack técnica

- **Proxy:** Nova Vercel function `api/b2b-scrape.js` (evita CORS)
- **Extração:** IA (OpenAI GPT) parseia o HTML e retorna JSON estruturado
- **Preview:** Reutilizar modal de importação existente
- **Persistência:** Mesmo fluxo de `mergeImportIntoBanco()` + Item Mestre

## Dev Notes

### Prompt para extração IA

```
Extraia os produtos com preços desta página de fornecedor.
Retorne APENAS um JSON array com objetos:
[{"nome": "...", "preco": 0.00, "unidade": "...", "marca": "..."}]
Se não encontrar produtos, retorne [].
Texto da página:
{text}
```

### b2b-scrape.js (Vercel function)

```javascript
export default async function handler(req, res) {
  if (req.method === "OPTIONS") return res.status(204).end();
  const { url } = req.body || {};
  if (!url) return res.status(400).json({ error: "URL required" });

  try {
    const r = await fetch(url, {
      headers: { "User-Agent": "Mozilla/5.0" },
      signal: AbortSignal.timeout(10000),
    });
    const html = await r.text();
    // Strip HTML tags, keep text
    const text = html
      .replace(/<script[^>]*>[\s\S]*?<\/script>/gi, "")
      .replace(/<style[^>]*>[\s\S]*?<\/style>/gi, "")
      .replace(/<[^>]+>/g, " ")
      .replace(/\s+/g, " ")
      .trim()
      .slice(0, 15000);
    const title = (html.match(/<title[^>]*>([^<]*)<\/title>/i) || [])[1] || "";
    res.status(200).json({ text, title, url });
  } catch (err) {
    res.status(500).json({ error: err.message });
  }
}
```
