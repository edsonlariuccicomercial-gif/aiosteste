# Story 2.6 — Design System Unificado

**Epic:** 2 — Resolucao de Debito Tecnico
**Sprint:** 3 (Semana 6-7)
**Prioridade:** P2 — ALTO
**Estimativa:** 20 horas
**Debitos resolvidos:** UX-02, UX-18, UX-20

---

## Descricao

Criar um design system unificado que substitua os dois sistemas visuais atuais (painel verde/escuro e GDP azul/slate). Definir design tokens compartilhados em CSS custom properties, criar componentes visuais base consistentes, e substituir emojis por icon library (Lucide). A base visual sera o tema azul/slate (GDP) por ser mais profissional e ter melhor contraste.

---

## Criterios de Aceitacao

- [ ] Arquivo `design-tokens.css` com todos os tokens de cor, tipografia, espacamento
- [ ] Todas as 7 paginas HTML importam `design-tokens.css`
- [ ] Tema base: azul/slate (dark) com accent azul (#3b82f6)
- [ ] Contraste minimo 4.5:1 para texto normal, 3:1 para texto grande (WCAG AA)
- [ ] Scale de espacamento consistente: 4, 8, 12, 16, 20, 24, 32, 40, 48, 64px
- [ ] Tipografia unificada: Inter como primaria, system-ui como fallback
- [ ] Componentes CSS base: `.btn`, `.badge`, `.card`, `.input`, `.table`
- [ ] Emojis substituidos por icones Lucide (SVG) em todas as paginas
- [ ] Visual regression tests (screenshots) para cada pagina antes/depois
- [ ] CSS inline nas paginas GDP reduzido em >= 50%
