## Story

**As a** fornecedor que envia pedidos ao Tiny/Olist,
**I want** que os SKUs dos itens do contrato sejam os códigos reais dos produtos já cadastrados no Tiny,
**so that** os pedidos referenciam produtos existentes e a unidade/NCM herda corretamente do cadastro.

---

## Root Cause (confirmado via Playwright + API)

O payload que enviamos ao Tiny está **correto** (KG, LT, NCM). Mas o Tiny usa o `codigo` do item do pedido para fazer match com um produto do cadastro. Se o `codigo` não existe no cadastro (ex: "990001"), o Tiny aceita o pedido mas não associa ao produto real — e a unidade/NCM pode ser ignorada ou sobrescrita.

Produtos reais no Tiny: `074` (Maçã KG), `076` (Macarrão Espaguete KG), etc. Nosso sistema gera códigos aleatórios que não existem no Tiny.

**Fix:** ao cadastrar itens no contrato, buscar no Tiny por nome e usar o `codigo` do produto existente. Não gerar códigos aleatórios.

## Acceptance Criteria

### AC-1: autoCadastrarTiny busca produtos existentes antes de criar novos
- [ ] Antes de gerar SKU aleatório, buscar no Tiny via `produtos.pesquisa.php` por descrição do item
- [ ] Se encontrar produto com nome similar: usar `codigo` e `unidade` do produto existente
- [ ] Se NÃO encontrar: criar produto novo (fluxo atual)
- [ ] Salvar o `codigo` real do Tiny como `item.sku` no contrato

### AC-2: tiny-produtos.js busca antes de criar
- [ ] Em `/api/tiny-produtos` (cadastro individual e batch): antes de `produto.incluir.php`, buscar por nome
- [ ] Se produto existe: retornar `sku` e `unidade` existentes (status: "existente")
- [ ] Se NÃO existe: criar normalmente com `normalizeUnit(item.unidade)`
- [ ] A busca usa `searchTinyProduct()` do order.js (reutilizar ou copiar a lógica)

### AC-3: Botão "Sincronizar com Tiny" nos detalhes do contrato
- [ ] Adicionar botão **"🔄 Sync Tiny"** nos detalhes do contrato
- [ ] Ao clicar: para CADA item, buscar no Tiny por nome → se encontrar, atualizar `sku` e `unidade` no contrato
- [ ] Toast: "X/Y itens sincronizados com Tiny"
- [ ] Util para contratos existentes que já têm SKUs aleatórios

### AC-4: order.js não gera SKU aleatório se possível
- [ ] No Step 1 do order.js: quando item não tem SKU e `searchTinyProduct` encontra match, usar o codigo existente
- [ ] Apenas gerar SKU aleatório como ÚLTIMO recurso (produto realmente inédito)

- [ ] 3.2 Implementar `sincronizarContratoTiny(contratoId)`:
  ```javascript
  async function sincronizarContratoTiny(contratoId) {
    const c = contratos.find(x => x.id === contratoId);
    if (!c) return;
    showToast(`Sincronizando ${c.itens.length} itens com Tiny...`, 3000);
    try {
      const resp = await fetch("/api/tiny-produtos", {
        method: "POST",
        headers: { "Content-Type": "application/json" },
        body: JSON.stringify({
          itens: c.itens.map(item => ({
            num: item.num, descricao: item.descricao, unidade: item.unidade,
            precoUnitario: item.precoUnitario, ncm: item.ncm || '',
            sku: '', codigo: ''  // Forçar busca no Tiny (não usar SKU local)
          })),
          contractId: contratoId, action: "cadastrar"
        })
      });
      const data = await resp.json();
      if (data.success && data.results) {
        let synced = 0;
        data.results.forEach(r => {
          if (r.sku) {
            const item = c.itens.find(it => it.num === r.num);
            if (item) {
              item.sku = r.sku;
              if (r.unidade) item.unidade = r.unidade;
              synced++;
              adicionarAoBancoProdutos(item);
            }
          }
        });
        saveContratos();
        abrirContrato(contratoId);
        showToast(`${synced}/${c.itens.length} itens sincronizados com Tiny!`, 4000);
      }
    } catch(err) {
      showToast("Erro: " + err.message, 4000);
    }
  }
  ```

### Fluxo corrigido
```
Import contrato → item "Maçã Fuji 1KG"
    ↓ autoCadastrarTiny
Tiny busca por "maca fuji" → encontra codigo: 074, unidade: KG
    ↓
item.sku = "074", item.unidade = "KG" (do Tiny real)
    ↓
Pedido criado → codigo: 074 (produto REAL do Tiny)
    ↓
Tiny: match com produto 074 → unidade KG herdada ✓
```
