### Arquivo XLS analisado (42 lancamentos)
```
Colunas: Cliente | Data Emissao | Data Vencimento | Valor documento | Situacao | Historico | Forma de recebimento | Meio de recebimento
Exemplo: Caixa Escolar Gabriel Toti | 31/10/2024 | 06/11/2024 | 698.36 | Em aberto | Ref. a NF n 268 | Cheque |
```

### Mapeamento XLS → Sistema
| Coluna XLS | Campo sistema | Tipo |
|-----------|--------------|------|
| Cliente | cliente | string |
| Data Emissao | dataEmissao | date DD/MM/YYYY |
| Data Vencimento | vencimento | date DD/MM/YYYY |
| Valor documento | valor | number |
| Situacao | status | "em_aberto" (mapear "Em aberto") |
| Historico | descricao | string |
| Forma de recebimento | forma | string (mapear: Boleto, Cheque, Ted, Pix) |
| Meio de recebimento | meioRecebimento | string (informativo) |

### Recomendacao: Importacao moderna (drag & drop)
Padrao usado por Stripe, QuickBooks, Notion:
1. Botao "Importar Planilha" ao lado de "+ Incluir"
2. Modal com zona drag & drop (aceita xls/xlsx/csv)
3. Parsing client-side via SheetJS (cdn.sheetjs.com/xlsx-latest/package/dist/xlsx.full.min.js)
4. Mapeamento automatico de colunas por nome (fuzzy match)
5. Preview com tabela mostrando primeiras 5 linhas
6. Botao "Importar N lancamentos" para confirmar
7. Nao precisa planilha modelo — aceita qualquer layout
