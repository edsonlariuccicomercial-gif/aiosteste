# Story 1.7 - Automacao da Rotina Operacional Diaria

## Status
Done

## Objetivo
Unificar a rotina operacional em um comando unico que execute validacoes essenciais, gere artefatos e publique um relatorio consolidado da rodada.

## Criterios de Aceite

- [x] Comando unico executa pipeline operacional padrao.
- [x] Relatorio JSON com status por etapa e duracao e gerado ao final.
- [x] Relatorio Markdown resumido e gerado para operacao.
- [x] Pipeline suporta execucao com e sem coleta SGD no inicio.

## Tarefas

- [x] Criar script orquestrador da rotina (`ops:daily`).
- [x] Encadear etapas de auditoria/snapshot/export/historico.
- [x] Persistir relatorio de execucao em JSON e Markdown.
- [x] Atualizar runbook com novo comando rapido.

## Riscos

- Falha em etapa unica interromper pipeline inteira sem diagnostico claro.
- Dependencia de credenciais no modo com coleta.

## Mitigacoes

- Relatorio detalhado por etapa com stderr resumido.
- Flag para pular coleta quando nao houver credencial ativa.

## Definition of Done

- [x] Criterios de aceite validados.
- [x] Story atualizada com evidencias reais.
- [x] File list atualizada.
