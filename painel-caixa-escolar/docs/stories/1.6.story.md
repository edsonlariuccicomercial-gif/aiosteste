# Story 1.6 - Sugestao de Lance com Base Historica e Urgencia

## Status
Done

## Objetivo
Gerar sugestao de lance operacional por cotacao combinando historico de preco por SKU/SRE, piso de preco minimo do modelo e fator de urgencia de prazo.

## Criterios de Aceite

- [x] Dashboard exibe coluna de `Lance recomendado`.
- [x] Sugestao considera historico (`SKU+SRE` com fallback para `SKU`) quando disponivel.
- [x] Sugestao nunca fica abaixo do `Preco minimo (modelo)`.
- [x] Regras de urgencia de prazo impactam o valor sugerido.
- [x] Exportacao CSV inclui campo de lance recomendado.

## Tarefas

- [x] Implementar calculo de lance recomendado no dashboard.
- [x] Exibir coluna nova na fila priorizada.
- [x] Integrar no CSV de exportacao filtrada/urgentes.
- [x] Atualizar documentacao da story com evidencias.

## Riscos

- Base historica pequena pode gerar sugestao enviesada.
- Parametros de urgencia podem reduzir agressivamente margem.

## Mitigacoes

- Aplicar piso minimo de preco obrigatorio.
- Expor formula simples e ajustavel por parametro.

## Definition of Done

- [x] Criterios de aceite validados.
- [x] Story atualizada com evidencias reais.
- [x] File list atualizada.

## Evidencias

- Dashboard atualizado com coluna `Lance recomendado`.
- Formula aplicada: referencia historica (mediana SKU+SRE/SKU) * fator de urgencia, com piso em `Preco minimo (modelo)`.
- Fatores de urgencia implementados: `<=2 dias => 0.97`, `<=5 dias => 0.985`, demais `1.0`.
- CSV de exportacao atualizado com campos `faixa_historica_sku` e `lance_recomendado`.
