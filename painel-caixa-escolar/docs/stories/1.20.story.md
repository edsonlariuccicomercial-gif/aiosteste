# Story 1.20 - Exibir Alertas Operacionais Ativos no Dashboard

## Status
Done

## Objetivo
Dar visibilidade imediata dos alertas operacionais no dashboard para reduzir tempo de reação em cenários `GO_WITH_WARNINGS` ou `NO-GO`.
