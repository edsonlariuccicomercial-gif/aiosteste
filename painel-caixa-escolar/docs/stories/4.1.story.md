# Story 4.1 — Dashboard Home Unificado com Controle de Acesso por Modulo

**Epic:** 4 — Evolucao Licit-IAX: Dashboard Unificado, Banco de Produtos e Lista de Compras
**Prioridade:** Alta
**Estimativa:** 24 horas
**Status:** Done

---

## Descricao

Criar pagina inicial (home/dashboard) do Licit-IAX que unifica as informacoes mais relevantes dos 3 modulos do sistema (Radar, Intel-Precos, GDP Contratos) com controle de acesso por modulo. Cada modulo sera vendido individualmente, entao o dashboard deve exibir apenas os cards/KPIs dos modulos que o usuario tem acesso liberado.

Inclui um globo 3D interativo com pontos coloridos representando as SREs (Superintendencias Regionais de Ensino) de atuacao da empresa. Ao clicar em um ponto, abre atalho para dados daquela regional.

**Referencia visual:** https://legiao-mu.vercel.app/guia — aba "Design", globo interativo abaixo do card "Titulos da Legiao".

### Restricoes

- NAO remover paginas existentes (index.html, gdp-contratos.html continuam funcionando)
- Home e um agregador, NAO duplicar funcionalidades dos modulos
- Globo deve ser leve (nao pode atrasar carregamento da pagina significativamente)
- Se Three.js for muito pesado, usar alternativa CSS 3D ou canvas 2D

### Opcoes de Implementacao do Globo

1. **Three.js** (recomendado) — globo 3D real, raycasting nativo, textura de mapa
2. **CSS 3D Transform** — leve, sem dependencia, limitado mas suficiente
3. **Canvas 2D** — meio termo, boa performance, sem 3D real
