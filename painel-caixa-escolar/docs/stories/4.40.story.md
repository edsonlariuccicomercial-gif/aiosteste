## Story

**As a** operador que navega entre múltiplas abas,
**I want** ter filtros consistentes (escola, município, status, data, texto) em todas as abas,
**so that** eu possa encontrar rapidamente o que procuro em qualquer lugar do sistema.

---

## Contexto / Motivação

Estado atual dos filtros por aba:

| Aba | Filtros existentes | Falta |
|-----|--------------------|-------|
| Radar (Oportunidades) | escola, município, grupo, status, SRE, texto | filtro por data |
| Envio SGD | escola, status, texto (Story 4.31) | filtro por data |
| Pré-Orçamento | nenhum | escola, status, texto, data |
| Aprovados/Histórico | nenhum | escola, status, texto, data |
| Banco de Preços | grupo, texto | — (suficiente) |

O objetivo é padronizar: mesmo layout visual, mesmas classes CSS, mesma estrutura HTML para os filtros em todas as abas que precisam deles.

## Dev Notes

### Função auxiliar de filtro por data

```javascript
function dentroDoIntervalo(dataStr, de, ate) {
  if (!dataStr) return true; // sem data: não filtrar
  const d = new Date(dataStr);
  if (de && d < new Date(de)) return false;
  if (ate && d > new Date(ate + 'T23:59:59')) return false;
  return true;
}
```
