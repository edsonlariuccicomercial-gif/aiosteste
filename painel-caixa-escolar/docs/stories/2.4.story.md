# Story 2.4 — Pipeline CI/CD com GitHub Actions

**Epic:** 2 — Resolucao de Debito Tecnico
**Sprint:** 2 (Semana 4-5)
**Prioridade:** P2 — ALTO
**Estimativa:** 12 horas
**Debitos resolvidos:** SYS-8, SYS-7 (parcial)

---

## Descricao

Criar pipeline de integracao continua com GitHub Actions que executa verificacoes automaticas em cada push e pull request. O pipeline deve incluir lint, testes unitarios, testes e2e, audit de seguranca e deploy automatico para Netlify na branch main.

---

## Criterios de Aceitacao

- [ ] Arquivo `.github/workflows/ci.yml` funcional
- [ ] Pipeline executa em cada push e pull request
- [ ] Pipeline inclui: lint (ESLint), testes unitarios (Vitest), audit de seguranca (npm audit)
- [ ] Pipeline inclui testes e2e (Playwright) em PRs para main
- [ ] Pipeline faz deploy automatico para Netlify quando merge em main
- [ ] Branch protection configurada: requer pipeline verde para merge
- [ ] `.eslintrc.js` configurado com regras basicas
- [ ] Dependencias CDN fixadas com versao especifica (SYS-7)
- [ ] Badge de status do pipeline no README

---

## Tarefas

- [ ] Criar `.github/workflows/ci.yml`
- [ ] Configurar job `lint` com ESLint
- [ ] Criar `.eslintrc.js` com regras para Vanilla JS (env: browser + node)
- [ ] Configurar job `test` com Vitest
- [ ] Configurar job `e2e` com Playwright (apenas em PRs para main)
- [ ] Configurar job `security` com `npm audit --audit-level=high`
- [ ] Configurar job `deploy` para Netlify (apenas em push para main)
- [ ] Configurar secrets no GitHub: `NETLIFY_AUTH_TOKEN`, `NETLIFY_SITE_ID`
- [ ] Configurar branch protection: require status checks
- [ ] Fixar versoes de SheetJS e PDF.js no HTML (CDN com versao)
- [ ] Adicionar badge de CI no README do projeto
- [ ] Testar pipeline com PR de teste

---

## Dependencias

- **Story 2.5** executa em paralelo — testes criados la sao executados aqui
- **GitHub:** Acesso admin para configurar branch protection e secrets

## Riscos

- Testes e2e requerem browser (Playwright) — GitHub Actions suporta, mas pode ser lento
- Free tier GitHub Actions: 2000 min/mes — suficiente para projeto pequeno

## Arquivos Afetados

- `.github/workflows/ci.yml` (novo)
- `.eslintrc.js` (novo)
- `package.json` (scripts lint, test)
- `squads/caixa-escolar/dashboard/index.html` (fixar versoes CDN)
- `README.md` (badge)
