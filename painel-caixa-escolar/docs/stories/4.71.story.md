## Story

**As a** gestor do GDP,
**I want** que os produtos cadastrados na Central de Produtos sincronizem com o Supabase,
**so that** os dados nao se perdem ao trocar de computador, limpar cache ou usar aba anonima.

---

## Acceptance Criteria

- [ ] **AC-1:** Chave `gdp.produtos.v1` adicionada ao array `GDP_SYNC_KEYS` em `gdp-core.js`
- [ ] **AC-2:** Chave `gdp.produtos.v1` adicionada ao array `SYNC_KEYS` em `app-sync.js` (se existir lista separada)
- [ ] **AC-3:** Ao salvar produto (saveBancoProdutos), dados sobem para Supabase via syncToCloud
- [ ] **AC-4:** Ao abrir o sistema em outro browser, produtos carregam do Supabase via syncFromCloud

---

## Bug Report

| Campo | Detalhe |
|-------|---------|
| **Erro** | Produtos nao sincronizam com Supabase — apenas localStorage |
| **Causa raiz** | `gdp.produtos.v1` nao esta no array `GDP_SYNC_KEYS` (gdp-core.js:117) |
| **Impacto** | 185 produtos do usuario existem apenas no browser atual |
| **saveBancoProdutos** | Chama `schedulCloudSync()` mas syncToCloud ignora a chave |
