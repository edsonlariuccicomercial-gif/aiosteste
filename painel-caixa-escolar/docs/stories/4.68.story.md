## Story

**As a** gestor financeiro do GDP,
**I want** que ao importar um extrato bancário, o sistema sugira automaticamente quais contas a pagar ou receber correspondem a cada lançamento pelo valor,
**so that** posso conciliar e baixar contas com um clique, sem precisar buscar manualmente em CP/CR.

---

## Acceptance Criteria

- [x] **AC-1:** Para cada lançamento pendente do extrato, o sistema busca automaticamente contas CP/CR com valor idêntico
- [x] **AC-2:** Débitos no extrato (valor < 0) buscam em Contas a Pagar pendentes
- [x] **AC-3:** Créditos no extrato (valor >= 0) buscam em Contas a Receber pendentes
- [x] **AC-4:** Sugestões aparecem na coluna Status com botão verde mostrando tipo (CP/CR) e descrição resumida
- [x] **AC-5:** Ao clicar na sugestão, o lançamento é conciliado E a conta é baixada automaticamente (status paga/recebida)
- [x] **AC-6:** Audit trail registra `baixaAutoConciliacao: true` e `extratoRef` na conta baixada
- [x] **AC-7:** Se não houver sugestão, o botão "Conciliar" manual continua disponível
- [x] **AC-8:** Lançamentos já conciliados mostram indicador de vínculo (CP) ou (CR)
- [x] **AC-9:** Múltiplas sugestões são ordenadas por proximidade de data (vencimento vs data do extrato)

---

## Regras de matching

| Condição extrato | Busca em | Filtro |
|-----------------|----------|--------|
| valor < 0 (débito/saída) | Contas a Pagar | status != "paga" AND valor == abs(extrato.valor) |
| valor >= 0 (crédito/entrada) | Contas a Receber | status != "recebida" AND valor == extrato.valor |

- Tolerância: R$ 0,01 (arredondamento)
- Ranking: proximidade de `vencimento` com `data` do extrato
- Sem tolerância de valor — matching exato para segurança

---

## Fluxo de conciliação com baixa

1. Usuário importa extrato (OFX/PDF/CSV)
2. Abre o extrato na lista
3. Para cada lançamento pendente, sistema mostra sugestão de CP/CR correspondente
4. Usuário clica na sugestão → concilia + baixa automática
5. Conta CP/CR muda status para paga/recebida com audit trail
6. KPIs de CP/CR atualizam automaticamente
