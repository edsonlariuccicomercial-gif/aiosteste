## Story

**As a** operador que ganhou uma licitação da Caixa Escolar,
**I want** marcar um pré-orçamento como "ganho" e criar automaticamente um contrato no módulo GDP,
**so that** o registro do contrato seja feito imediatamente, sem duplicar o trabalho de preenchimento manual.

---

## Contexto / Motivação

Quando o operador vence uma licitação da Caixa Escolar, ele precisa:

1. Registrar que o pré-orçamento foi ganho (status no Caixa Escolar)
2. Criar um contrato no módulo GDP (Gestão de Pedidos) com os dados do fornecimento

Hoje essas duas etapas são completamente manuais e desconectadas: o pré-orçamento fica em "enviado" e o operador abre o GDP para criar o contrato do zero. Os dados (escola, itens, valor, SRE) já estão no pré-orçamento — basta transferi-los para o GDP automaticamente.

### Estrutura do GDP

Os contratos do GDP são armazenados em localStorage na chave `gdp.contratos.v1` como array de objetos. O arquivo `gdp-contratos.html` é o módulo que os lê e exibe. A estrutura exata do contrato precisa ser verificada no código do gdp-contratos para garantir compatibilidade.

- [ ] Ao confirmar, cria um objeto contrato no formato esperado por `gdp-contratos.html`
- [ ] Campos mapeados do pré-orçamento:
  - `numeroContrato` → valor digitado no modal

- [ ] Pré-orçamentos sem status "ganho" continuam funcionando normalmente

### Função de criação do contrato GDP

```javascript
function criarContratoGdp(preOrc, { numeroContrato, dataAssinatura }) {
  const chave = 'gdp.contratos.v1';
  const contratos = JSON.parse(localStorage.getItem(chave) || '[]');

  const novoContrato = {
    id: crypto.randomUUID(),
    numeroContrato,
    escola: preOrc.escola,
    municipio: preOrc.municipio,
    sre: preOrc.sre,
    itens: preOrc.itens.map(item => ({
      nome: item.nome || item.descricao,
      quantidade: item.quantidade,
      valorUnitario: item.precoUnitario || item.custo,
      unidade: item.unidade || 'UN'
    })),
    valorTotal: preOrc.itens.reduce((s, i) => s + (i.precoUnitario || i.custo || 0) * (i.quantidade || 1), 0),
    dataAssinatura: dataAssinatura || new Date().toISOString().split('T')[0],
    status: 'ativo',
    origem: 'caixa-escolar',
    preOrcId: preOrc.id,
    dataCriacao: new Date().toISOString()
  };

  contratos.push(novoContrato);
  localStorage.setItem(chave, JSON.stringify(contratos));
  return novoContrato;
}
```
