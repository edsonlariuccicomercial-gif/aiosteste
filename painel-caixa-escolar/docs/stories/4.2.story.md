# Story 4.2 — Banco de Produtos ERP (GDP Contratos)

**Epic:** 4 — Evolucao Licit-IAX: Dashboard Unificado, Banco de Produtos e Lista de Compras
**Prioridade:** Alta
**Estimativa:** 12 horas
**Status:** Done

---

## Descricao

Criar nova aba "Banco de Produtos ERP" em `gdp-contratos.html` para gerenciar um catalogo de produtos simplificado. O banco armazena apenas descricao, codigo (SKU) e unidade. Permite inclusao manual e importacao via Excel e PDF.

Este banco servira como fonte de dados para sincronizacao com itens de contratos (Story 4.3).

- [ ] Aba "Banco de Produtos" adicionada na navegacao horizontal de `gdp-contratos.html`
- [ ] Aba posicionada apos "Entregas" (ultima aba atual)
- [ ] Aba ativa exibe tabela de produtos com colunas: #, Descricao, Codigo (SKU), Unidade, Acoes

- [ ] Botao "+ Novo Produto" abre formulario/modal com campos: Descricao, Codigo (SKU), Unidade

### Importacao via Excel

- [ ] Botao "Importar Excel" abre file picker para .xlsx/.xls/.csv

- [ ] Botao "Importar PDF" abre file picker para .pdf
- [ ] Utiliza API `ai-parse-price.js` existente para extrair texto do PDF
- [ ] Parser AI extrai descricao, SKU e unidade do texto

### Persistencia

- [ ] Dados salvos em localStorage com chave `gdp.produtos.v1`
- [ ] Formato: `{ updatedAt: ISO, itens: [{ id, descricao, sku, unidade, criadoEm }] }`

- [ ] Busca/filtro por texto na tabela de produtos

### Acoes na Tabela

- [ ] Botao excluir: confirmacao antes de remover
