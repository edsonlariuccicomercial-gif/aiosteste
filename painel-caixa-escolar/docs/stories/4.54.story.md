## Story

**As a** gestor do sistema GDP,
**I want** filtros de período nas telas financeiras e pedidos, e a lista de extratos na conciliação conforme o modelo visual,
**so that** eu consiga filtrar dados por dia/semana/mês/intervalo em todas as telas e ver os extratos importados no formato correto.

---

## Acceptance Criteria

### AC-1 — Filtro por período em Contas a Pagar
- [ ] Ao lado do dropdown de categoria existente (`cp-filtro-categoria`), adicionar filtro por período: Todos, Hoje, Esta Semana, Este Mês, Intervalo
- [ ] Ao selecionar "Intervalo", mostrar 2 inputs date (de/até)
- [ ] Filtrar pela data de vencimento (`item.vencimento`)
- [ ] Combinar com busca e categoria existentes
- [ ] Footer deve refletir dados filtrados (já implementado na 4.52)
- [ ] Arquivo HTML: `gdp-contratos.html` — seção contas-pagar
- [ ] Arquivo JS: `gdp-pedidos.js` → `renderContasPagar()`

### AC-2 — Filtro por período em Contas a Receber
- [ ] Mesma lógica de AC-1 aplicada a Contas a Receber
- [ ] Filtrar pela data de vencimento (`item.vencimento`)
- [ ] Arquivo HTML: `gdp-contratos.html` — seção contas-receber
- [ ] Arquivo JS: `gdp-pedidos.js` → `renderContasReceber()`

### AC-3 — Atualizar filtro de Pedidos para período (Hoje/Semana/Mês/Intervalo)
- [ ] O filtro de data existente nos pedidos (`filtro-data-pedido`) deve ser substituído/atualizado para o mesmo modelo: Todos, Hoje, Esta Semana, Este Mês, Intervalo
- [ ] Ao selecionar "Intervalo", mostrar 2 inputs date (de/até)
- [ ] Filtrar pela data do pedido (`p.data` ou `p.dataEntrega`)
- [ ] Arquivo HTML: `gdp-contratos.html` — seção pedidos
- [ ] Arquivo JS: `gdp-pedidos.js` → `renderPedidos()`

### AC-4 — Layout da lista de extratos na conciliação conforme modelo
- [ ] A lista de extratos importados na aba Conciliação Bancária deve seguir o layout da imagem de referência:
  - Fundo escuro (dark theme, consistente com o sistema)
  - Tabela com colunas: checkbox (para seleção), Data (com ícone ordenação), Arquivo (com ícone ordenação), Conta financeira (com ícone ordenação), Conciliados/Total
  - Bordas sutis entre header e rows
  - Header com texto em cinza claro e ícones de sort (⇅)
  - Rows com texto branco
  - Checkbox alinhado à esquerda
- [ ] Quando não há extratos: não mostrar tabela
- [ ] O extrato salvo deve ser exibido em formato FECHADO (colapsado) — ao clicar no nome do arquivo, abre a conciliação
- [ ] Referência visual: imagem `extrato.jpg`
