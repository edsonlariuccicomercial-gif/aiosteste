## Story

**Como** operador do Licit-AIX,
**Quero** um motor de precificação que calcule preço de venda usando Mark-up multiplicador (com frete, tributos, custos indiretos), gere 3 cenários (Conservador/Sugerido/Agressivo) baseados em Should-Cost Analysis, e adapte a estratégia pela classificação Kraljic do produto,
**Para que** eu precifique com fundamentação técnica, compliance fiscal e inteligência competitiva em vez de margem fixa arbitrária.

---

### Motor Mark-up
- [ ] **AC3:** Cálculo usa fórmula: `Preço = Custo × (1 / (1 - (Frete% + CI% + Tributos% + Margem%)))`
- [ ] **AC4:** Custo vem do menor custo válido em `custos_fornecedores` (Camada 2)
- [ ] **AC5:** Se produto tem `frete_estimado` específico, usa esse; senão usa `frete_padrao_pct` da config
- [ ] **AC6:** Mark-up exibido na cotação (ex: "Mark-up: 1.515×")

### 3 Cenários (Should-Cost Analysis / McKinsey)
- [ ] **AC7:** Cenário **Conservador** (Does-Cost): margem = margem_desejada + 5% → proteção
- [ ] **AC8:** Cenário **Sugerido** (Should-Cost): margem calibrada pela mediana de preços vencedores do histórico (Camada 4)
- [ ] **AC9:** Cenário **Agressivo** (Could-Cost): margem = margem_minima → competitividade máxima
- [ ] **AC10:** Cada cenário mostra: preço, mark-up, margem líquida %, probabilidade de ganho %, risco (Baixo/Médio/Alto)

### Influência Kraljic
- [ ] **AC11:** Produto "alavancagem" → cenário default = Agressivo
- [ ] **AC12:** Produto "gargalo" → cenário default = Conservador
- [ ] **AC13:** Produto "estratégico" → cenário default = Sugerido
- [ ] **AC14:** Produto "não-crítico" → cenário default = Agressivo

### Indicadores
- [ ] **AC15:** **Price-to-Win**: preço estimado para ganhar (mediana vencedores na região)
- [ ] **AC16:** **Competitiveness Index**: nosso preço / mediana vencedores (< 1.0 = competitivo)

## Dev Notes

### Fórmula Mark-up:
```
mark_up = 1 / (1 - (frete_pct + ci_pct + tributos_pct + margem_pct))
preco_venda = custo_unitario × mark_up

Exemplo:
  Custo: R$ 18.00 | Frete: 5% | CI: 8% | Tributos: 6% | Margem: 15%
  mark_up = 1 / (1 - 0.34) = 1.515
  preco = 18.00 × 1.515 = R$ 27.27
```

### Cenário "Sugerido" — calibração pelo histórico:
```
mediana_vencedores = mediana(historico_licitacoes.preco_vencedor) 
                     filtrado por produto + região
margem_calibrada = (mediana_vencedores - custo_tco) / mediana_vencedores
preco_sugerido = custo × mark_up(margem = margem_calibrada)
```

### Probabilidade de ganho:
```
prob_ganho = count(historico onde nosso_preco <= preco_vencedor) / total_historico × 100
```

### Configuração empresa — campos:
```javascript
{
  regime_tributario: "simples",      // simples | presumido | real
  aliquota_tributos_pct: 6.0,       // % sobre faturamento
  custos_indiretos_pct: 8.0,        // admin, aluguel, depreciação
  frete_padrao_pct: 5.0,            // frete médio
  margem_minima_pct: 5.0,           // mínimo aceitável
  margem_desejada_pct: 15.0         // target normal
}
```
