## Descricao

Com os dados fluindo para `preco_historico` (Story 8.9) e as funcoes RPC de agregacao disponiveis (Story 8.10), esta story cria o dashboard visual de serie historica. O operador podera consultar precos medios por produto/municipio, visualizar tendencias temporais, filtrar por SKU/SRE/periodo, e ver KPIs de competitividade.

A UI sera uma nova secao no GDP (aba ou sub-aba na Central de Produtos), seguindo o design system existente.

**Referencia SOP:** Etapa 14 — Serie Historica (visualizacao e analytics)

### Dados via RPC

```javascript
// Carregar tabela de precos medios
async function carregarSerieHistorica(filtros = {}) {
  const cacheKey = `serie-hist-${JSON.stringify(filtros)}`;
  const cached = getCachedData(cacheKey, 15 * 60 * 1000); // 15 min TTL
  if (cached) return cached;

  const { data, error } = await supabase.rpc('preco_medio_por_municipio', {
    p_empresa_id: empresaAtual.id,
    p_meses: filtros.meses || 6,
  });

  if (data) setCachedData(cacheKey, data);
  return data || [];
}

// Carregar tendencia mensal
async function carregarTendencia(sku, meses = 12) {
  const { data, error } = await supabase.rpc('tendencia_preco_mensal', {
    p_empresa_id: empresaAtual.id,
    p_sku: sku,
    p_meses: meses,
  });
  return data || [];
}
```

### KPIs

```javascript
function calcularKPIs(dados) {
  const totalGanhos = dados.reduce((s, d) => s + d.total_ganhos, 0);
  const totalPerdidos = dados.reduce((s, d) => s + d.total_perdidos, 0);
  const taxaGeral = totalGanhos / (totalGanhos + totalPerdidos) * 100;

  const porSku = agruparPor(dados, 'sku');
  const competitivos = Object.entries(porSku)
    .map(([sku, items]) => ({
      sku,
      taxa: items.reduce((s, d) => s + d.total_ganhos, 0) /
            items.reduce((s, d) => s + d.total_ganhos + d.total_perdidos, 0) * 100,
    }))
    .sort((a, b) => b.taxa - a.taxa);

  return {
    taxaGeral,
    top3Competitivos: competitivos.slice(0, 3),
    top3Risco: competitivos.reverse().slice(0, 3),
  };
}
```
