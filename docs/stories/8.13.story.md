## Story

**Como** operador do painel caixa escolar,
**Quero** que as notas fiscais de entrada permitam importar os produtos diretamente para a Central de Produtos e disponibilizem a NF para download em PDF,
**Para que** eu possa alimentar meu catálogo automaticamente a partir de notas reais e manter um arquivo das NFs para consulta/auditoria.

---

## Acceptance Criteria

### Importação de Produtos
- [x] AC1: Na visualização de uma NF de entrada, existe botão "Importar Produtos para Central"
- [x] AC2: Ao clicar, cada item da NF é mapeado para a Central de Produtos (nome, descrição, NCM, unidade, custo)
- [x] AC3: Se o produto já existe na Central (match por NCM ou nome), o sistema atualiza o custo
- [x] AC4: Se o produto é novo, cria registro na Central com dados da NF preenchidos
- [x] AC5: Após importação, feedback visual mostra quantos produtos foram importados vs atualizados vs ignorados

### Download PDF da NF
- [x] AC6: Na listagem de NFs de entrada, cada NF tem ícone/botão de download PDF
- [x] AC7: O PDF gerado contém todos os dados da NF (emitente, destinatário, itens, totais, impostos)
- [x] AC8: Se a NF original é XML, o sistema renderiza em formato visual antes de gerar o PDF
- [x] AC9: O download funciona offline (dados já em localStorage/Supabase)

---

## Scope

### IN
- Botão de importação de produtos da NF para Central
- Lógica de match/dedup por NCM e nome
- Geração de PDF a partir de dados da NF (XML ou JSON)
- Download do PDF no navegador

### OUT
- Emissão de NF (já existe em outro fluxo)
- Importação automática sem confirmação do usuário
- Integração com sistemas externos de NF
