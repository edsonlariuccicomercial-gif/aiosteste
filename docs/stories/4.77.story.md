# Story 4.77 — Reescrever generateDanfePdf server-side com layout DANFE completo (jsPDF vetorial)

- **Epic:** EPIC-4 — GDP Caixa Escolar
- **Status:** Done
- **Complexity:** M (Medium)
- **Agent:** @dev
- **Depends on:** Story 4.75
- **Priority:** Critical (PDF do email ilegível — rasterizado por html2pdf.js)

## Descrição

O PDF DANFE enviado no email é gerado por `html2pdf.js` que rasteriza o HTML como imagem bitmap e cola no PDF — resultado: borrado, pesado, artefatos. O "Visualizar DANFE" é perfeito porque o browser renderiza nativamente.

A solução é reescrever `generateDanfePdf()` em `send-order-email.js` (server-side, jsPDF) para gerar um DANFE vetorial completo replicando fielmente o layout: recibo, header com NF/série, chave de acesso, emitente, destinatário, impostos, produtos, dados adicionais, footer.

No client-side, remover a dependência de `html2pdf.js` para gerar o PDF — usar SOMENTE o PDF server-side (jsPDF vetorial) como anexo.

## Acceptance Criteria

### AC1: PDF DANFE vetorial completo no servidor
- **Given** o email é enviado com NF-e
- **When** o servidor gera o PDF
- **Then** o PDF tem layout DANFE completo: recibo, header (NF número, série, DANFE), chave de acesso, protocolo, emitente, destinatário, impostos, tabela de produtos, dados adicionais, footer
- **And** texto é selecionável (vetorial, não imagem)

### AC2: Client-side não gera mais PDF via html2pdf
- **Given** o email está sendo enviado
- **When** o payload é montado
- **Then** `danfePdf` NÃO é gerado client-side via `gerarDanfePdfBase64()`
- **And** o servidor sempre gera o PDF com `generateDanfePdf()`

### AC3: Layout fiel ao Visualizar DANFE
- **Given** o PDF é gerado
- **When** aberto no leitor de PDF
- **Then** contém todas as seções da DANFE real conforme `abrirDanfeNotaFiscal()`

## Scope

### IN
- Reescrever `generateDanfePdf()` em `send-order-email.js` com layout DANFE completo
- Remover geração client-side de PDF (`gerarDanfePdfBase64` call) dos payloads de email
- Servidor SEMPRE gera o PDF quando há `nfe` no payload

### OUT
- Mudanças no "Visualizar DANFE" (funciona perfeito)
- Mudanças no layout do email texto
