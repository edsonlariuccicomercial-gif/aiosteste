# Story 8.3 — Deprecar Banco de Precos — Facade + Atualizar RadarMatcher

**Epic:** Epic 8 — Consolidacao GDP e Fechamento de Gaps Operacionais
**Status:** Ready
**Priority:** P1
**Story Points:** 2

---

## Descricao

Apos a migracao de dados (Story 8.2), o Banco de Precos (`banco-precos-client.js`) ainda e referenciado em varios pontos do sistema: `RadarMatcher`, `alimentarBancoComResultado()`, e a aba "Banco de Precos" no dashboard. Esta story converte o SDK `BancoPrecos` em uma facade que redireciona todas as chamadas para a Central de Produtos unificada, e atualiza o `RadarMatcher` para buscar na Central.

**Referencia SOP:** Etapas 6-7 (Unificacao) + Etapa 8 (Matching via RadarMatcher)

---

## Criterios de Aceitacao

- [ ] AC-1: SDK `BancoPrecos` em `banco-precos-client.js` funciona como facade — chamadas existentes (`calcularPreco()`, etc.) internamente consultam `gdp.produtos.v1`
- [ ] AC-2: `alimentarBancoComResultado()` em `app-results.js:667` grava na Central unificada (`gdp.produtos.v1`) em vez de `caixaescolar.banco.v1`
- [ ] AC-3: `RadarMatcher` em `radar-matcher.js` busca produtos na Central unificada (`gdp.produtos.v1`) em vez do Banco de Precos
- [ ] AC-4: Aba "Banco de Precos" no dashboard e removida ou redirecionada para "Central de Produtos"
- [ ] AC-5: `renderBanco()` em `app-banco.js` e substituido por chamada a `renderBancoProdutos()` enriquecido
- [ ] AC-6: API externa `cotacoes-lariucci.vercel.app` continua acessivel via SDK como feature opcional (CON-G2-002)

## Notas Tecnicas

### Facade Pattern

O SDK `BancoPrecos` em `banco-precos-client.js` sera refatorado assim:

```javascript
class BancoPrecos {
  // Antes: lia de caixaescolar.banco.v1
  // Depois: le de gdp.produtos.v1 (Central unificada)

  static getItem(nome) {
    const central = JSON.parse(localStorage.getItem('gdp.produtos.v1') || '{}');
    return (central.itens || []).find(p =>
      normalizarNome(p.descricao) === normalizarNome(nome)
    );
  }

  static calcularPreco(nome, quantidade) {
    const produto = this.getItem(nome);
    if (!produto || !produto.precoReferencia) return null;
    return produto.precoReferencia * quantidade;
  }

  // API externa continua opcional
  static async consultarPrecoRemoto(nome) {

  }
}
```

### RadarMatcher — Atualizacao de Fonte

- **Arquivo:** `radar-matcher.js`
- **Atual:** `RadarMatcher.match()` busca em `bancoPrecos.itens` (de `caixaescolar.banco.v1`)
- **Novo:** `RadarMatcher.match()` busca em `bancoProdutos.itens` (de `gdp.produtos.v1`)
- **Impacto:** O matcher ganha acesso a campos extras (SKU, NCM, unidade, marca) que melhoram o vinculo

### alimentarBancoComResultado()

- **Arquivo:** `app-results.js` linha 667
- **Atual:** Grava resultado em `caixaescolar.banco.v1`
- **Novo:** Grava resultado nos campos `historicoResultados`, `propostas` do produto correspondente em `gdp.produtos.v1`

### Aba Dashboard

- **Arquivo:** `index.html` (ref. linha 1346 do carregamento de `banco-precos-client.js`)
- **Arquivo:** `app-banco.js` (funcao `renderBanco()`)
- **Acao:** Remover aba "Banco de Precos" do dashboard ou redirecionar para "Central de Produtos"
