# Story 11.1 — GDP Visual Refresh: Tabela Flat de Pedidos

**Epic:** 11 — GDP Visual Refresh
**Status:** Ready for Review
**Complexidade:** SIMPLE (6/25)
**Agente executor:** @dev (Dex)

## Objetivo

Transformar a apresentacao visual da tab Pedidos do GDP de estilo "card" para estilo "tabela flat" com separadores finos, inspirado no layout do ERP Olist. Somente aparencia — zero mudanca funcional.

## Acceptance Criteria

### AC1: Tabela flat (sem card container)
- [x] `#tab-pedidos .table-wrap` sem background, border e border-radius
- [x] Tabela usa `border-collapse: separate` com `border-spacing: 0 2px`
- [x] Linhas da tabela com fundo transparente

### AC2: Headers da tabela com estilo muted
- [x] `th` sem `text-transform: uppercase` e sem `letter-spacing`
- [x] `th` com `font-weight: 400`, `font-size: .82rem`, `color: var(--mut)`
- [x] `th` com `padding: 10px 6px`
- [x] `th` com `border-bottom: 1px solid var(--bdr)` separando header do corpo

### AC3: Celulas da tabela com padding reduzido
- [x] `td` com `padding: 9px 6px`, `font-size: .88rem`
- [x] `td` com `border-bottom: 1px solid rgba(71,85,105,.15)` (separador sutil)
- [x] `tr:hover td` com `background: rgba(78,201,138,.04)`

### AC4: Search bar ampliado
- [x] Input de busca com `height: 40px`
- [x] `border-radius: 6px`, sem borda visivel, `background: var(--bg)`
- [x] Icone de lupa posicionado a esquerda do input
- [x] Placeholder descritivo: "Pesquise por cliente, CNPJ ou contrato..."
- [x] Largura flexivel (`flex: 1; max-width: 500px`)

### AC5: Status tabs com dots coloridos
- [x] Tabs renderizados como elementos inline (nao botoes estilo btn)
- [x] Cada tab com: dot colorido (8x8px circle) + nome do status + contador
- [x] Tab ativo com `border-bottom: 2px solid var(--accent)` e `font-weight: 600`
- [x] Tabs inativos com `opacity: .7`
- [x] Contador abaixo do nome (font-size .78rem, cor muted)

### AC6: Rodape de totais alinhado
- [x] Totais (quantidade + valor) dentro de `<tfoot>` da tabela
- [x] Alinhamento com colunas correspondentes
- [x] Labels "quantidade" / "valor total (R$)" em texto muted abaixo dos valores

### AC7: Zero regressao funcional
- [x] Checkbox de selecao funciona normalmente
- [x] Menu de acoes (...) funciona normalmente
- [x] Filtros (busca, data, contrato) funcionam normalmente
- [x] Bulk actions funcionam normalmente
- [x] Navegacao para detalhe do pedido funciona normalmente
- [x] Status tabs filtram corretamente

1. **CSS escopado** — Usar seletores `#tab-pedidos .table-wrap`, `#tab-pedidos th`, `#tab-pedidos td` para nao afetar outras tabs
2. **JS incremental** — Alterar `renderPedidosStatusTabs()` para novo formato de tabs, ajustar footer para usar `<tfoot>`
3. **HTML minimo** — Apenas o search bar precisa de mudanca estrutural
