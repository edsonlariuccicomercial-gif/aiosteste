## Story

**Como** operador do Licit-AIX,
**Quero** que ao importar Excel/PDF/NF os preços sejam gravados na tabela `custos_fornecedores` separada (não sobrescrevam o produto),
**Para que** eu tenha múltiplos registros de custo por fornecedor × data × região, com campos TCO (frete, prazo pagamento, confiabilidade).

---

## Acceptance Criteria

- [x] **AC1:** Import de Excel grava em `custos_fornecedores` com `origem: "excel"` e `confiabilidade: 0.95`
- [x] **AC2:** Import de PDF (texto) grava com `origem: "pdf"` e `confiabilidade: 0.85`
- [x] **AC3:** Import de PDF (OCR) grava com `origem: "pdf-ocr"` e `confiabilidade: 0.50`
- [x] **AC4:** Produto existente na central NÃO é sobrescrito — apenas novo registro de custo é adicionado
- [x] **AC5:** Produto novo (não existe na central) é criado automaticamente + primeiro custo registrado
- [x] **AC6:** Campo `fornecedor` capturado do nome do arquivo ou campo mapeado na importação
- [x] **AC7:** Campo `frete_estimado` disponível no mapeamento de colunas (opcional)
- [x] **AC8:** Campo `prazo_pagamento_dias` disponível no mapeamento de colunas (opcional)
- [x] **AC9:** Preview de importação mostra quantos custos novos serão adicionados vs produtos existentes
- [x] **AC10:** Após import, modal de custos do produto mostra os novos registros com badge de origem
