## Story

**Como** operador do Licit-AIX,
**Quero** um dashboard de inteligência competitiva baseado no histórico de licitações, com métricas de win rate, price-to-win, concorrentes frequentes e tendências por SRE,
**Para que** eu tome decisões de precificação baseadas em dados reais e não em intuição.

## Dev Notes

### Tabela historico_licitacoes — estrutura:
```javascript
{
  id: "HIST-timestamp-random",
  escola: "EE Exemplo",
  cidade: "Uberaba",
  sre: "uberaba",
  produto_id: "PROD-xxx",        // FK central_produtos
  descricao_item: "Arroz tipo 1 5kg",
  preco_proposto: 27.50,          // nosso preço
  preco_vencedor: 25.00,          // preço do vencedor
  empresa_vencedora: "Concorrente ABC",
  participou: true,
  ganhou: false,
  motivo_perda: "preco",          // preco | tecnico | desclassificado | outro
  delta_percent: 10.0,            // (nosso - vencedor) / vencedor × 100
  data: "2025-03-15",
  orcamento_sgd_id: "SGD-123"
}
```

### Fórmulas das métricas (§1.1-G):
- **Win Rate** = ganhos / (ganhos + perdidos) × 100
- **Price-to-Win** = mediana(preco_vencedor) por produto × região
- **Competitiveness Index** = nosso_preço / mediana_vencedores
- **Margin Erosion** = Δ% margem média entre trimestres
